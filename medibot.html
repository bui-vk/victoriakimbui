Software Engineering: Self-Driving Medication Delivery Robot (Medibot)

About this ongoing project

The Medibot project is an ongoing initiative that involves the development of a self-driving medication delivery robot using a Turtlebot equipped with a 2D LiDAR and a camera for delivering medication stored in an attached box to patients.

The Medibot is designed to feature an LED display, lights, and a speaker to signal delivery/order status. It operates through a web application where staff place orders for medication delivery from one station to another.

Key Features:

  • Self-driving capabilities
  • Obstacle avoidance and lane-keeping
  • LED display, lights, and speaker for delivery/order status
  • Utilization of Python, ROS, and Ubuntu commands
  • Documentation via Confluence and ticket management using Jira
  • Collaborative work involving 24 people
  • Ongoing project focused on development and refinement

The development involves eight teams: Camera, Art, SLAM (Simultaneous Localization and Mapping), Navigation, Driving Platform, App, Fancy Hardware, and Advanced Tech, totaling 24 people, working collaboratively on various aspects of the project.

This ongoing project aims to continue its development and refinement to achieve its objectives.
